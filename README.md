# Simulation of Web Series Genre Preferences Among Youth

## Overview

This project analyzes web series genre preferences among youth using a dataset. The objective is to identify trends, patterns, and user interests through data visualization.

---

## Objectives

* Analyze genre popularity among youth
* Understand viewing trends
* Represent data using visualizations
* Extract meaningful insights

---

## Tools and Technologies

* Python
* Pandas
* Matplotlib
* Jupyter Notebook

---

## Project Structure

```
├── netflix_titles.xlsx      # Dataset  
├── analysis.ipynb           # Code file  
├── report.docx              # Project report  
├── colab_output.pdf         # Output file   
```

---

## Features

* Data cleaning and preprocessing
* Multiple visualizations (bar graphs, pie charts, etc.)
* Clear and structured analysis
* Insight generation from dataset

---


## Conclusion

The project analyzes web series genre preferences and highlights key trends among youth using data visualization techniques.
